## What is QWICC?
[QWICC]([url](https://www.sparkfun.com/qwiic)) is a connector system which offers a standarized, 4-pin, JST connector
to easily create systems.
<img width="200" height="200" alt="image" src="https://github.com/user-attachments/assets/60a5cc16-b231-44aa-bbf8-3e9ddd62cb66" />

Note: QWIIC is compatible with [Adafruit's StemmaQT System]([url](https://learn.adafruit.com/introducing-adafruit-stemma-qt/sparkfun-qwiic)) and this way even more components are available.

### Sparkfun

Sparkfun offers a bunch of sensors, displays, buttons with QWIIC Interface. 
All QWIIC components from Sparkfun can be found here: 
[QWIIC Special on Sparkfun]([url](https://www.sparkfun.com/special-categories/qwiic.html))
